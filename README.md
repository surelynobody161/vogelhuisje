# Nestly React-Native app

## Get started

1. Install dependencies

   ```bash
   npm install
   ```
   
2. Setup .env
Duplicate the .env.example file and rename it to .env
   ```bash
   API_URL=http://145.24.223.199:8085
   ```

3. Start the app

   ```bash
   npx expo start
   ```
   
You can now scan the QR code to view the app.
